KIPRCopter  1.1
Our API is designed to work on the CBC to control the AR Drone, however, our repository includes project files to be directly imported into an xcode project on a mac and allows a person to connect to and fly the drone. The video stream will not work at this time because it is currently integrated with the CBC however future plans are to utilize the drone's on board detection system to perform autonomous functions.

Parameters
enableA value indicating if movement is enabled. 0 - True 1 - False
pitchA value from zero to one indicating the percentage of maximum tilt in the left or right direction negative values are left and positive values are right. Ex: -.5 means Half of the total tilt left
rollA value from zero to one indicating the percentage of maximum tilt in the forward or backward direction negative values are left and positive values are right. Ex: -.5 means Half of the total tilt backwards.
yawA value indicating the rotational velocity of the dronein milieters per second
gazA value indicating the change in altitude in milimeters per second
